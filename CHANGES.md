## Changes
Removal of whitespace
```
#Python
def main():
    print("Hello")
```
```
#PyPlus
def main()
{
    print("Hello")
}
```

Renaming of booleans
```
#Python
flse = False
tre = True
```
```
#PyPlus
flse = false
tre = true
```

Renaming of None
```
#Python
myVar = None
```
```
#PyPlus
myVar = null
```
